# Object-Detection
Built a real-time Object Detection system using YOLO and OpenCV.
🎯 Objective

To build a reliable object detection system that can accurately identify bottles, showcasing real-world applications of AI in automation and visual recognition tasks.

📚 Learning Outcomes
Gained hands-on experience in computer vision workflows
Understood model training, evaluation, and optimization
Improved knowledge in object detection architectures like YOLO
Worked on real-time video processing using OpenCV

🚀 Future Improvements
Improve detection accuracy with larger datasets
Extend detection to multiple object classes
Deploy as a web or mobile application
